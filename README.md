# ADiSC (under construction)
ADiSC- Automatic Distinction of SPECI Criteria
## Overview
-ADiSC is a Python tool to detect and identify if weather changes meet SPECI criteria for reports of aviation weather observations. 

-The SPECI criteria varies depending on airport. So ADiSC employs a method where users set criteria, and the system determines whether or not SPECI is necessary based on those criteria.

-This tool may be useful for beginners of aviation weather observers/reporters and it also may be able to contribute to automating them.
